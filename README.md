# Smart Student Tracking System

A web-based student tracking and attendance monitoring system developed using Python Flask as part of MCA project learning and practical web application development.

The project helps monitor student login activity, location tracking, attendance workflow, and admin approval management through a simple dashboard interface.

---

## Features

* Student login and logout system
* Basic GPS location tracking
* Admin approval workflow
* Real-time dashboard monitoring
* Attendance activity tracking
* Interactive map integration
* Responsive user interface
* Student status monitoring
* Cloud deployment using Render

---

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript
* Leaflet Maps
* GitHub
* Render Deployment

---

## Project Purpose

This project was developed to understand:

* Web application development
* Flask backend workflow
* Database integration
* Real-time tracking concepts
* Dashboard management
* Deployment using GitHub and Render

It also helped improve practical learning, problem-solving, and project implementation skills.
