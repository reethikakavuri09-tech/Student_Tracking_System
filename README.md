# Smart Student Tracking System

A real-time GPS-based student tracking and attendance monitoring system built using Flask, Python, SQLite, JavaScript, and Leaflet Maps.

## Features
- Live GPS location tracking
- Student attendance monitoring
- Admin approval workflow
- Real-time dashboard
- Interactive map integration
- Login/logout tracking
- Responsive UI

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS/JavaScript
- Leaflet Maps
- Render Deployment
- GitHub
